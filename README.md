## About

Mobile Application Developer pursuing MCA at MES Institute of Management and Career Courses, Pune. Currently interning at **Worxmate** building production-grade React Native applications with real-world responsibilities — from push notification architecture to app store releases on both **Google Play** and the **Apple App Store**.

My primary focus is cross-platform mobile development using **Flutter** and **React Native**, with hands-on experience in performance optimization, state management via BLoC, and Firebase integration.
